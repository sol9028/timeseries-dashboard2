다변량 시계열 이상탐지 대시보드

CSV 기반 시계열 데이터를 자동 분석하여 이상 탐지, 통합 판정, 성능 평가, 수동 보정, 결과 다운로드까지 제공하는 Streamlit 대시보드입니다.

프로젝트 개요

본 프로젝트는 강의 14에서 다룬 Darts 기반 시계열 이상탐지 구조를 Streamlit 웹 대시보드로 구현한 프로젝트입니다.

강의에서 다룬 기본 구조인

Scorer → Detector → Aggregator

흐름을 기반으로 하되, 실제 사용자가 CSV 파일을 업로드하여 분석할 수 있도록 다음 기능을 확장했습니다.

CSV 업로드
→ 컬럼 자동 추정 및 선택
→ Darts TimeSeries 변환
→ Scorer 기반 이상 점수 계산
→ Detector 기반 이상 플래그 변환
→ Aggregator 기반 최종 이상 판정
→ 성능 평가 및 수동 보정
→ 결과 CSV 다운로드

본 대시보드는 특정 데이터셋에 고정된 분석 코드가 아니라, 사용자가 업로드한 임의의 다변량 시계열 CSV에 대해 이상탐지 파이프라인을 적용할 수 있도록 구성했습니다.

⸻

주요 특징

1. CSV 기반 다변량 시계열 자동 처리

사용자가 CSV 파일을 업로드하면 시간 컬럼, 값 컬럼, 라벨 컬럼을 선택할 수 있습니다.

시간 컬럼은 timestamp, time, date, datetime 등의 이름을 기준으로 자동 추정되며, 값 컬럼은 수치형 변수 중에서 선택할 수 있습니다.

라벨 컬럼은 선택 사항입니다. 라벨이 있는 경우 정량 평가를 수행하고, 라벨이 없는 경우 라벨-프리 진단 지표를 제공합니다.

⸻

2. Darts 기본 Scorer와 PyOD 확장 Scorer

Darts의 기본 Scorer를 사용하여 예측 오차와 분포 변화 기반 이상 점수를 계산합니다.

사용 가능한 Darts Scorer는 다음과 같습니다.

NormScorer
KMeansScorer
WassersteinScorer

또한 PyOD 기반 비지도 이상탐지기를 Darts Scorer처럼 사용할 수 있도록 확장했습니다.

사용 가능한 PyOD Scorer는 다음과 같습니다.

LOF
COPOD
ECOD

이를 통해 단순 예측 오차뿐만 아니라 밀도 기반 이상, 분포 기반 이상, 경험적 CDF 기반 이상까지 함께 탐지할 수 있습니다.

⸻

3. Detector 기반 이상 플래그 변환

각 Scorer가 계산한 이상 점수는 QuantileDetector를 통해 정상/이상 이진 플래그로 변환됩니다.

Detector 임계 분위수를 조정하면 탐지 민감도를 변경할 수 있습니다.

임계 분위수 높음 → 더 보수적 탐지
임계 분위수 낮음 → 더 민감한 탐지

⸻

4. Aggregator 기반 결과 통합

여러 Scorer의 탐지 결과를 하나의 최종 이상 플래그로 통합합니다.

지원하는 통합 방식은 다음과 같습니다.

OR
- 하나의 Scorer라도 이상이라고 판단하면 최종 이상
AND
- 모든 Scorer가 이상이라고 판단해야 최종 이상
다수결 MajorityVoteAggregator
- 설정한 합의 수준 이상이 이상이라고 판단하면 최종 이상

MajorityVoteAggregator는 aggregators.py에서 직접 구현한 기능입니다.
Darts의 Aggregator 구조를 확장하여, OR과 AND 사이의 민감도를 사용자가 조절할 수 있도록 했습니다.

예를 들어 Scorer가 3개이고 합의 수준이 0.50이면, 3개 중 2개 이상이 이상이라고 판단한 경우 최종 이상으로 확정합니다.

⸻

5. 라벨 유무별 평가 대시보드

정답 라벨 컬럼이 있는 경우 다음 평가 지표를 제공합니다.

AUC-ROC
AUC-PR
Precision
Recall
F1-score
Confusion Matrix
ROC Curve
PR Curve

정답 라벨이 없는 경우에는 다음과 같은 라벨-프리 진단 결과를 제공합니다.

이상 탐지 비율
이상 점수 분포
임계값 민감도
Scorer 간 일치도

이를 통해 라벨이 없는 실제 데이터에서도 탐지 결과가 과도하거나 부족하지 않은지 확인할 수 있습니다.

⸻

6. 클릭 기반 수동 보정

자동 탐지 결과를 사용자가 직접 수정할 수 있습니다.

차트에서 특정 시점을 클릭하거나 드래그로 선택한 뒤, 다음 작업을 적용할 수 있습니다.

이상 해제
- 자동으로 이상이라고 탐지된 지점을 정상으로 변경
이상으로 추가
- 자동 탐지되지 않은 지점을 이상으로 직접 추가

수동 보정 결과는 자동 탐지 결과와 구분되어 표시되며, 최종 다운로드 CSV에도 반영됩니다.

또한 잘못 적용한 보정을 되돌릴 수 있도록 수동 보정 초기화 기능을 제공합니다.

⸻

7. 결과 CSV 다운로드

탐지 결과는 CSV 파일로 다운로드할 수 있습니다.

다운로드 파일에는 다음 정보가 포함됩니다.

timestamp
Scorer별 이상 점수
Scorer별 이상 플래그
Aggregator 통합 결과
수동 보정 반영 최종 결과
수동 보정 여부
정답 라벨

이를 통해 대시보드에서 확인한 이상탐지 결과를 외부 분석이나 보고서 작성에 활용할 수 있습니다.

샘플 데이터

발표 및 시연에는 교통 센서 상황을 가정한 합성 다변량 시계열 데이터를 사용할 수 있습니다.

샘플 데이터 예시:

sample_multivariate_anomaly.csv

샘플 데이터 컬럼은 다음과 같습니다.

컬럼명	설명
timestamp	5분 간격 관측 시점
occupancy	도로 점유율 또는 혼잡도 지표
speed	평균 이동 속도
is_anomaly	이상 여부 라벨, 정상=0 / 이상=1

샘플 데이터는 실제 교통 센서 데이터의 구조를 참고하여 만든 합성 데이터입니다.
정상 구간에는 하루 주기와 출퇴근 시간대 패턴이 반영되어 있으며, 일부 구간에는 occupancy 급증 또는 speed 급락 같은 이상 패턴이 포함되어 있습니다.

⸻

CSV 입력 형식

업로드할 CSV는 최소한 다음 요소를 포함해야 합니다.

항목	설명	필수 여부
시간 컬럼	datetime으로 변환 가능한 컬럼	필수
값 컬럼	이상탐지 대상 수치형 변수 1개 이상	필수
라벨 컬럼	정상=0, 이상=1 형태의 정답 라벨	선택

예시:

timestamp,occupancy,speed,is_anomaly
2015-09-01 00:00:00,10.2,58.4,0
2015-09-01 00:05:00,9.8,59.1,0
2015-09-01 00:10:00,11.5,56.7,1


⸻

권장 실행 순서

1. 샘플 CSV 파일을 업로드합니다.
2. 시간 컬럼으로 timestamp를 선택합니다.
3. 값 컬럼으로 occupancy, speed를 선택합니다.
4. 라벨 컬럼이 있다면 is_anomaly를 선택합니다.
5. 기본 Scorer 설정으로 이상탐지를 실행합니다.
6. 데이터 개요에서 기초 통계와 시계열 패턴을 확인합니다.
7. 이상탐지 결과에서 Scorer별 이상 점수와 탐지 구간을 확인합니다.
8. 통합 이상 플래그에서 Aggregator 결과를 확인합니다.
9. 평가 대시보드에서 AUC-ROC, AUC-PR, Precision, Recall, F1을 확인합니다.
10. 필요하면 수동 보정에서 오탐을 제거하거나 이상을 추가합니다.
11. 최종 결과 CSV를 다운로드합니다.

본 프로젝트는 강의에서 다룬 Darts 기반 시계열 이상탐지 구조를 실제 사용 가능한 Streamlit 대시보드로 확장한 것입니다. 사용자는 CSV 파일을 업로드하고 시간 컬럼, 값 컬럼, 라벨 컬럼을 선택하면 자동으로 이상 점수 계산, 이상 판정, Scorer 결과 통합, 성능 평가, 수동 보정, 결과 다운로드까지 수행할 수 있습니다. 또한 Darts 기본 Scorer에 더해 PyOD 기반 Scorer를 추가하고, 직접 구현한 MajorityVoteAggregator를 통해 여러 Scorer의 결과를 하나의 최종 이상 판정으로 통합했습니다.
